## üå± Beginner Track (0-3 months)

### Month 1: Foundations

#### Week 1-2: Understanding AI Basics

**Goal**: Understand what AI is and how it works

**Resources**:
- [Elements of AI](https://www.elementsofai.com/) - Free, no coding required
- [AI For Everyone by Andrew Ng](https://www.coursera.org/learn/ai-for-everyone) - Coursera
- [Introduction to AI - Google](https://www.coursera.org/learn/google-introduction-to-ai)

**Activities**:
- Complete Elements of AI Part 1
- Watch: "What is AI?" videos on YouTube
- Read: [What is vibe coding?](https://theconversation.com/what-is-vibe-coding-a-computer-scientist-explains-what-it-means-to-have-ai-write-computer-code-and-what-risks-that-can-entail-257172)

**Checkpoint**: Can you explain AI, ML, and deep learning to a friend?

#### Week 3-4: First Steps with AI Coding Tools

**Goal**: Build your first project with AI assistance

**Resources**:
- [Bolt.new](https://bolt.new/) - No coding required
- [v0 by Vercel](https://v0.dev/chat) - For UI components
- [Replit](https://replit.com/) - Interactive coding

**Activities**:
- Build a simple landing page using Bolt.new
- Create a to-do app with Replit
- Experiment with v0 for UI components

**Project**: Personal portfolio website using AI tools

### Month 2: Learning to Code with AI

#### Week 1-2: Python Fundamentals

**Goal**: Learn basic programming with AI assistance

**Resources**:
- [Python for Everybody](https://www.py4e.com/) - Free course
- [Codecademy Python](https://www.codecademy.com/learn/learn-python-3)
- Use [GitHub Copilot](https://github.com/features/copilot) or [Cursor](https://www.cursor.com/)

**Activities**:
- Complete Python basics course
- Write simple scripts with AI assistance
- Learn to read and understand AI-generated code

**Checkpoint**: Can you write and modify basic Python scripts?

#### Week 3-4: Prompt Engineering Basics

**Goal**: Learn to communicate effectively with AI

**Resources**:
- [Learn Prompt Engineering](https://www.codecademy.com/learn/learn-prompt-engineering)
- [The Prompt Engineering Playbook](https://addyo.substack.com/p/the-prompt-engineering-playbook-for)
- [GitHub Copilot Chat Cookbook](https://docs.github.com/en/copilot/tutorials/copilot-chat-cookbook)

**Activities**:
- Practice writing clear prompts
- Learn prompt patterns (explain, refactor, debug)
- Document your best prompts

**Project**: Build a calculator app using AI with well-crafted prompts

### Month 3: Building Real Projects

#### Week 1-2: Web Development Basics

**Goal**: Understand HTML, CSS, JavaScript with AI

**Resources**:
- [freeCodeCamp](https://www.freecodecamp.org/) - Responsive Web Design
- [The Odin Project](https://www.theodinproject.com/)
- Use AI tools for learning and building

**Activities**:
- Build responsive web pages
- Learn JavaScript basics
- Use AI to explain concepts

**Checkpoint**: Can you build a multi-page website?

#### Week 3-4: First Full-Stack Project

**Goal**: Build and deploy a complete application

**Resources**:
- [Vibe Coding 101 with Replit](https://www.deeplearning.ai/short-courses/vibe-coding-101-with-replit/)
- [Lovable](https://lovable.dev/) or [Bolt.new](https://bolt.new/)

**Activities**:
- Plan a project (blog, portfolio, tool)
- Build with AI assistance
- Deploy to production

**Final Project**: Full-stack web application deployed online

## üìñ Study Tips

### Effective Learning Strategies

#### 1. Active Learning
- **Build while learning**: Don't just watch tutorials
- **Explain concepts**: Teach others what you learn
- **Debug actively**: Understand errors, don't just fix them

#### 2. Consistent Practice
- **Daily coding**: Even 30 minutes helps
- **Weekend projects**: Dedicate time for bigger projects
- **Code reviews**: Review others' code regularly

#### 3. Community Engagement
- **Join communities**: Reddit, Discord, forums
- **Ask questions**: No question is too basic
- **Share progress**: Blog, Twitter, LinkedIn

#### 4. Using AI Effectively
- **Don't copy blindly**: Understand AI-generated code
- **Ask for explanations**: Use AI to learn, not just code
- **Iterate prompts**: Refine your questions
- **Verify outputs**: Always test and review

### Common Pitfalls to Avoid

‚ùå **Over-reliance on AI**: Always understand the code
‚ùå **Tutorial hell**: Build projects, don't just watch
‚ùå **Skipping fundamentals**: Master basics first
‚ùå **Ignoring errors**: Learn from mistakes
‚ùå **Isolation**: Engage with the community

### Recommended Study Schedule

#### Weekday (2 hours)
- 30 min: Theory/tutorials
- 60 min: Hands-on coding
- 30 min: Review and documentation

#### Weekend (4-6 hours)
- 1 hour: Review week's learning
- 3-4 hours: Project work
- 1 hour: Community engagement
